# dotfiles

## .vimrc
usage:
```
$ cp vimrc ~/.vimrc
```

if you use patagonia color scheme
```
# clone patagoniavim
$ git clone https://github.com/cjgajard/patagonia-vim.git

# copy a color scheme file
$ mkdir ~/.vim/colors
$ cp patagonia-vim/colors/* ~/.vim/colors/
```
